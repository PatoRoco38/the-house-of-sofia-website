# The House of Sofia Website

Official institutional website for **The House of Sofia**.

This project presents the institutional identity, method positioning, governance structure and official communication channels of The House of Sofia.

## Overview

The website was designed as a private, elegant and institutional digital presence.

Its purpose is not to operate as a public booking system, sales page or treatment catalog.  
It serves as an official institutional entry point for presenting:

- The House concept
- Sofia Method
- Governance principles
- Concession model
- Institutional access
- Official contact channel

Audio System

The website includes an institutional audio layer based on the Sofia sound direction:

Sofia Threshold — initial entry sound
Sofia Site Loop — continuous ambient loop
Sofia Pulse — short interaction sound for strategic buttons

The audio system respects browser autoplay restrictions and starts only after user interaction.
